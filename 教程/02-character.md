# 角色设计与人物一致性

## 方法一：LLM生成描述 → 即梦生成图片

### Step 1：用豆包生成角色提示词

将剧本文档上传给豆包，提示词：
> 我是一个角色设计师。
> 我需要为上传的文档中的剧本设计人物角色。
> 要求详细设计，包含人物的外貌、发型、发色、体型、穿着等造型内容。
> 请认真阅读文档，结合故事背景，帮我生成适用于**即梦AI图片生成**功能的人物角色提示词。

输出：每个角色一段详细描述（外貌+服饰+即梦提示词格式）。

### Step 2：在即梦中生成角色图片

**设置：**
- 模式：图片生成
- 模型：图片4.0
- 比例：9:16（角色单图）或 16:9（三视图）
- 分辨率：2K（练习够用）

**提示词结构：**
```
[风格描述] [角色特征描述] [姿势/构图要求]
```

示例：
```
3D风格，迪士尼动画风格，融入中式美学，简单纯色背景，三视图，
[粘贴豆包生成的角色特征描述]，
人物保持双臂自然下垂，正面站立
```

**关键约束：**
- `双臂自然下垂，正面站立`——后期图生图保持一致性的基础，**必须加**
- 生成三视图便于后续参考

## 方法二：参考图 → LLM → 即梦生成

适合有风格偏好或参考形象的场景（从花瓣等设计网站找参考图）。

**步骤：**
1. 在豆包的**图像生成**功能中，通过参考图按钮上传3张参考图
2. 提示词加入：
   ```
   Q版3D风格，迪士尼动画风格，融入中式美学，
   参考以上图片风格，
   [角色特征描述]，
   人物保持双臂自然下垂，正面站立，
   背景：[场景描述]
   ```
3. 比例：9:16
4. 不满意可重新生成（豆包免费，可多次刷新）

---

## 保持角色一致性（图生图）

已有满意的角色参考图后，通过**图生图**生成不同动态：

**操作：**
1. 模式：图片生成（即梦）
2. 输入框左下角的带加号小框 → 上传角色参考图
3. **修改提示词**：
   - 保留：风格描述 + 角色特征描述
   - 删除：`双臂自然下垂`、`正面站立`、`三视图`、`纯色背景`
   - 替换动作描述：加一句引导语

**引导语模板：**
```
参考图中的人物形象，生成[场景+动作描述]
```

示例：
```
参考图中的人物形象，生成少女在树林里奔跑的画面，背景是在山林中，周围有许多的植物
```

**效果：** 同一角色形象 + 不同动态 + 不同背景，人物一致性良好。
