# 图片生成

## 一、文生图提示词公式

### 提示词 6 要素

| 要素 | 说明 | 示例 |
|------|------|------|
| 主体元素 | 画面核心视觉构成 | `树下草坪上摆着一只大西瓜` |
| 背景环境 | 烘托主体的场景信息 | `西瓜周围开了很多花朵，晴朗的天空` |
| 视觉风格 | 美术风格类型 | `插画风格` |
| 色彩调控 | 色调情绪 | `暖色调，高饱和，橙绿系主色调` |
| 文字构成 | 海报文字内容 | `"夏日大瓜" 手写体，冰块材质` |
| 氛围用语 | 画面整体情绪 | `清新夏日氛围，悠闲惬意` |

### 完整示例

```
主体：树下草坪上摆着一只大西瓜
背景：西瓜周围开了很多花朵，草地，树木，晴朗的天空
风格：插画风格
色调：暖色调，高饱和，橙绿系主色调
文字："夏日大瓜" 手写体，木质结构
氛围：清新夏日氛围，悠闲惬意
```

### 参数设置

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 模型 | 即梦 4.0 / 4.1 | 版本标号越高效果越好 |
| 比例 | 16:9 / 9:16 / 2:3 | 电脑平台 16:9；手机平台 9:16 或 2:3 |
| 分辨率 | 2K / 4K | 4K 消耗更多点数，细节更丰富 |

### 文字效果增强

1. 点击输入框下方**文字效果增强**按钮
2. 自动添加双引号，输入框内输入文字内容
3. 文字会以艺术字体形式出现在画面中
4. 可指定字体材质：`用冰做成的字体`、`木质结构`

## 二、参考图生图

### 1. 修改颜色

**操作：**
1. 上传参考图到预备窗口
2. 提示词：`将人物的头发修改为暗红色，其他部分保持不变`

### 2. 修改环境

**操作：**
1. 上传参考图
2. 提示词：`给人物的背景更换为健身房，只改变背景不改变人物`

### 3. 修改姿势和角度

**操作：**
1. 上传参考图
2. 提示词：`人物在咖啡厅喝咖啡的画面`

### 4. 固定造型修改环境（电商产品）

**操作：**
1. 上传产品参考图
2. 点击图片 → 智能参考 → 抠图 → 保存（锁定主体）
3. 提示词：`固定产品的造型，给产品更换一个沙漠的背景，背景中有金字塔和河流`

### 5. 参考风格

**操作：**
1. 上传风格参考图
2. 提示词：`参考图片的风格，生成配色姿势设计不同的图片`

### 6. ControlNet 功能（3.0 模型）

| 检测类型 | 用途 | 说明 |
|---------|------|------|
| 边缘轮廓 | 锁定线稿轮廓 | 适合保持造型设计不变 |
| 景深检测 | 修改风景季节 | 过时功能，建议直接用 4.0 模型提示词 |
| 人物姿势 (OpenPose) | 锁定人物骨骼姿势 | 适合保持姿势不变换装换人 |

**操作：**
1. 模型切换为**即梦 3.0**
2. 点击参考图 → 选择检测类型（边缘轮廓/景深/人物姿势）
3. 提示词描述修改内容
4. 调整参考强度（默认即可）

### 7. 多图参考融合

**操作：**
1. 模型选择**即梦 4.0**
2. 依次上传多张参考图（衣服图 + 人物图）
3. 提示词：`让图 2 的人物穿上图 1 的衣服`
4. 比例选择与参考图一致或使用智能比例

## 三、Agent 模式

Agent 模式集成即梦所有功能，通过自然语言理解需求，一步完成多步骤任务。

### 1. 一键生成连贯视频

**提示词示例：**
```
先生成一组图片，保持造型前后的一致性，分别表现学生起床、学生吃早餐、学生到学校门口、学生走进校园、学生走进教室，比例 16:9，并使用智能多帧功能生成连贯的视频。
```

### 2. 动态壁纸生成

**操作：**
1. 上传参考壁纸图
2. 提示词：`将图片制作为动态壁纸`

### 3. 治愈系动画生成

**提示词示例：**
```
以参考图为开头，分别生成森林木屋、河流桥梁、湖泊等画面，风格与参考图保持一致，最后将这些图片用智能多帧功能生成为视频。
```

### 4. 组图生长视频

**操作：**
1. 依次上传 4 张图片
2. 提示词：`将这一组图片生成为连贯的人物在水下游泳的视频`

### 5. 字体设计动态视频

**提示词示例：**
```
根据参考图的排版和文字位置，首先生成字体分别为"夏日狂欢"、"秋高气爽"、"冰雪世界"的海报，注意海报风格和组成字体的元素要契合内容所表述的季节，然后将包括参考图在内的 4 张图片使用智能多帧功能生成为一条连贯的视频。
```

### 6. 海报产品替换

**操作：**
1. 上传完整海报图 + 新产品图
2. 提示词：`将图 1 中的耳机更换为图 2 中的电子表，相应的产品名称、产品特点介绍部分的文字也进行与产品相关的更换`
3. 支持咖啡机、手表等多种产品替换

### 7. 产品试穿

**操作：**
1. 上传产品图（如耳环）
2. 提示词：`生成一个模特近景特写试穿参考图中的耳环`
3. 多产品试穿：上传多个产品图，提示词 `生成两个模特分别试穿参考图中的两个耳环`

### 8. 创意宣传海报

**提示词示例：**
```
根据产品参考图，生成该产品的创意宣传海报，要求包含产品名称特点介绍，同时设计上最好充满童趣，与产品本身的设计相搭配。
```

## 四、AI 角色设计

### 角色设计要素

| 要素 | 内容 |
|------|------|
| 整体造型 | 年龄、性别、性格 |
| 服装 | 上衣、下装、鞋子、配饰 |
| 发型 | 样式、颜色、头发装饰 |
| 画风 | 风格（写实/二次元）、配色 |

### 角色一致性控制方法

**方法 1：提示词控制**
```
生成一个角色的正面视角，全身，手臂自然下垂，站立姿态的角色设计提示词。
从以下几个方面来书写：
造型包括年龄性别性格；
服装包括上衣下装鞋子配饰；
发型包括样式颜色装饰；
美术风格包括风格和配色。
要求简洁而具体，背景使用白色简单的背景。
```

**方法 2：参考图生图（更稳定）**
1. 生成全身正面角色图（白色背景）
2. 下载满意结果
3. 上传为参考图
4. 提示词只描述场景和动作：`人物在咖啡厅喝咖啡`
5. 删除视角、姿势、背景相关描述词

### 生成视频片段

1. 点击满意图片 → 生成视频
2. 提示词描述运镜和行为：`镜头拉近，向镜头打招呼`
3. 时长 5 秒

## 五、图片生成视频

### 1. 首帧图生视频

**提示词结构：**
```
运镜描述 + 画面事物变化
```

**示例：**
```
固定镜头，人物戴上墨镜
```
```
镜头跟随，父亲骑着三轮车带着儿子行驶在乡间小道，风吹动麦田和路边的杂草
```

### 2. 首尾帧生视频

**用途：** 精确控制视频结尾画面

**操作：**
1. 首帧上传起始图
2. 尾帧上传目标图（用参考图生图生成）
3. 提示词：`固定镜头，人物戴上墨镜`

### 3. 智能多帧生视频

**用途：** 一次性生成多行为连贯视频

**操作：**
1. 切换为智能多帧模式
2. 依次上传 3-5 张图片（首帧→中间帧→尾帧）
3. 图片间添加衔接提示词：`人物拿出汉堡`、`人物吃汉堡`
4. 生成时长根据分镜数量调整（4 分镜建议 10 秒）

### 4. 主体参考生视频（不推荐）

当前模型表现一般，人物造型和服装款式易变形，环境倾向于原图而非提示词描述。

## 六、电影风格多分镜生成

### 新海诚风格示例

**Step 1：豆包 AI 生成图片提示词**
```
你是一个提示词书写大师，给我三个可以生成新海诚电影风格的中文提示词，要求包括主体、背景、风格、色调、氛围的描述。
```

**Step 2：即梦生成图片**
1. 复制提示词到即梦
2. 比例 16:9，模型 4.0
3. 下载满意结果

**Step 3：豆包 AI 生成运镜提示词**
```
请以这张图片作为起始镜头，打造一个多机位视频，呈现出壮观的氛围。需要 4 个连贯分镜，分镜头脚本结构为：景别 + 视角 + 运镜 + 画面内容，每个分镜用不同的运镜方式，要求用词简洁，字数不超过 400 字。
```

**Step 4：即梦生成视频**
1. 上传首帧图
2. 粘贴运镜提示词
3. 时长 5-10 秒（根据分镜数量）

## 八、爆款创意视频生成

**流程：**
1. 即梦首页搜索"萌宠"，复制喜欢的作品提示词
2. 豆包 AI：`请帮我根据上述提示词，再编写 5 条萌宠创意图片的提示词`

4. 视频提示词：`固定镜头，狗充满好奇的看着镜头`

### 2. 萌宠走秀类

**视频提示词：**
```
镜头跟随，主体向前走
```

### 3. 微缩场景类

**即梦首页搜索**：`微观`

**视频提示词：**
```
固定镜头，上方的小人在挤抹茶酱，下方拿着铲子的小人在用铲子铲奥利奥碎
```

## 九、真人与动漫合照

### 流程

**Step 1：分割转绘**
1. 上传真人照片 + 动漫风格参考图
2. 提示词：`将图 1 的风格转换为和图 2 相同的风格，其他部分保持不变`
3. 下载转绘结果

**Step 2：合照合成**
1. 上传真人原图 + 转绘图（或知名动漫角色图）
2. 即梦首页搜索"动漫合照"复制提示词模板
3. 修改性别、穿着等描述
4. 比例根据姿势选择（站姿 9:16）
5. 生成后使用智能超清修复边缘

**Step 3：视频生成**
1. 提示词：`固定镜头，两个人物向镜头招手`
