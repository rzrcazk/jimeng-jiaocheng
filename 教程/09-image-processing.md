# 图像处理

## 一、智能超清

**用途：** 提升图片分辨率，增加细节清晰度。

**操作：**
1. 鼠标悬置图片 → 点击**智能超清**
2. 参数设置：
   - 图片类型：自动 / 场景 / 人像
   - 参考程度：
     - 原图保持程度（越高越接近原图）
     - 细节生成程度（越高细节越多，但可能失真）
   - 放大倍率：2K→4K
3. 点击生成

**效果对比：**
- 放大后可拖拽分割线查看前后对比
- 模糊部分高清化，增加画面细节

**建议：** 保持默认参数即可，过高设置可能导致失真。

---

## 二、局部重绘

**用途：** 修改画面中局部不满意的区域，保留其他部分不变。

**操作：**
1. 点击图片 → 查看大图 → 局部重绘
2. 选区笔工具：圈选需要修改的区域
3. 橡皮擦工具：擦除多选区域
4. 移动工具：移动视图方便选中边角
5. 提示词描述修改内容：`将眼睛修改为绿色`
6. 点击生成

**技巧：**
- 描述具体颜色：`低饱和度的绿色`、`偏蓝一些的绿色`
- 选区精确度影响生成效果

---

## 三、扩图功能

**用途：** 扩展画面边界，补充画面外内容。

**操作：**
1. 点击图片 → 查看大图 → 扩图
2. 拖拽选框调整扩充范围和位置
3. 选择扩图比例：1:1 / 3:4 / 9:16 / 4:3 / 16:9
4. 提示词描述扩充内容（可选，不填则基于原图内容生成）
5. 点击开始生成

**典型应用：**
- 半身像扩展为全身像（成功率高）
- 横屏扩展为竖屏

---

## 四、消除笔

**用途：** 消除画面中不需要的物体或水印。

**操作：**
1. 点击图片 → 查看大图 → 消除笔
2. 调节画笔大小
3. 拖拽选中需要消除的内容（人物、水印等）
4. 点击生成

**注意：**
- 可消除画面中的人物和水印
- 即梦生成图片自带水印无法用此功能消除
- 需要专门的去水印工具处理

---

## 五、细节修复

**用途：** 原本针对 1K 分辨率图片优化，现不推荐使用。

**现状：**
- 当前即梦生成图片最低 2K 分辨率
- 细节修复功能未及时更新
- 可能产生负面效果（弱化原有细节）

**建议：** 使用**智能超清**替代
