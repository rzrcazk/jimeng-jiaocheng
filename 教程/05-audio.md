# 配音与配乐

## 一、AI 配乐

### 方法 1：剪映音乐库（最快）

1. 剪映 → 右上角**音频** → 音乐库
2. 左侧按类型筛选，或顶部搜索框输入关键词（如"欢快古典"、"史诗"）
3. 点击预览，满意后拖到下方音频轨道
4. 在时间轴末端分割，删除多余部分，对齐视频长度
5. ⚠️ 商用注意版权问题

### 方法 2：AI 生成配乐（魔兔 AI）

#### Step 1：用豆包生成配乐提示词

上传故事梗概文档，提示词：
> 我是一位 AI 配乐大师，擅长制作各种类型的音乐。
> 现在需要根据上传的文本，创作一个与故事内容、时代氛围契合的**纯音乐**。
> 请仔细阅读故事文本，为我生成用于 AI 生成的提示词。
> 提示词需要提供明确的音乐风格描述词（例如流行、摇滚、史诗等），
> **请用一段话进行表述，纯音乐，字数控制在 500 字以内。**

#### Step 2：魔兔 AI 生成

1. 打开魔兔 AI → AI 音乐
2. 开启**纯音乐**开关
3. 将豆包生成的提示词粘贴到**音乐风格**输入框
4. 填写歌名（可让 LLM 生成）
5. 选择最高编号的大模型
6. 点击开始生成 → 满意则下载，不满意可重新生成
7. 积分耗尽可换其他平台（Suno、Udio 等逻辑相同）

### 方法 3：即梦 AI 配乐

**操作：**
1. 视频生成页面 → 右下方**AI 配乐**
2. 两种模式：
   - 根据画面配乐：自动分析画面内容
   - 自定义配乐：选择场景/流派/情感/乐器
3. 示例设置：场景=销售，流派=轻音乐，情感=随意，乐器=小提琴
4. 生成 3 段配乐供选择

## 二、AI 配音

### 平台：海螺 AI（MiniMax 语音）

打开海螺 AI → 点击左上角**语音** → 进入 MiniMax 语音操作界面

### 配音类型选择

| 类型 | 用途 |
|------|------|
| 新闻播报 | 标准播报风格，适合正式内容 |
| 说书 | 有叙述感，适合故事旁白 |
| 影视配音 | 带情感起伏，适合角色对白 |

### 参数设置

- 模型：选最高版本（如 2.5）
- 音色库筛选：语言 → 口音 → 性别 → 年龄
- 情绪：旁白选"自动"；对白可选"开心"/"愤怒"等
- 语速：用滑块调整（历史叙事风格适当调慢）

### 旁白生成流程

1. 选择**说书**类型
2. 选择合适音色（如标准普通话男声，中年）
3. 粘贴旁白文本
4. 点击生成音频 → 预览 → 调整语速/情绪 → 重新生成
5. 下载音频

### 角色对白生成流程

1. 选择**影视配音**类型
2. 根据角色选择匹配音色（女孩选甜美女声等）
3. 输入角色台词
4. 添加情绪标注（如"开心"）
5. 生成、下载

## 三、对口型（即梦数字人）

将角色图片与配音合成为对口型视频片段。

### 操作方式

1. 即梦 → 模式切换到**数字人**
2. 输入框左侧第一个小框 → 上传角色图片（静态图）
3. 第三个按钮 → **上传音频**（上传已生成的配音文件）
4. 输入框中描述画面动作：
   ```
   小女孩伸手接过飘落的树叶，树叶飘落在手上
   ```
5. 点击发送 → 获得角色对口型视频片段

### 模式选择

| 模式 | 特点 | 用途 |
|------|------|------|
| 快速模式 | 生成速度快，成本低 | 快速预览 |
| 生动模式 | 默认，平衡速度与质量 | 常规使用 |
| 大师模式 | 口型匹配效果最佳，画质更好 | 成片输出 |

### 音色调节

- 音色选择：点击音色框，弹出音色选择窗口
- 说话速度：部分音色支持调节
- 情感调节：部分音色支持（开心/愤怒/悲伤等）
  - 示例：妩媚女生音色 + 开心情绪 → 更生动的演绎

### 动作描述

**用途：** 让静态图片人物在对口型时做出指定动作

**提示词示例：**
```
镜头拉远，人物比剪刀手
```
```
固定镜头，人物说台词，风吹动人物的衣服
```

### 图片 vs 视频参考

| 参考类型 | 效果 | 建议 |
|---------|------|------|
| 静态图片 | 口型匹配自然，面部肌肉活动流畅，身体动作自然 | 推荐使用 |
| 视频参考 | 效果略逊一筹，口型开合幅度与动态协调性稍差 | 不推荐 |

### 消耗积分

- 数字人配音：2 积分（省去 1 个视频生成积分）
- 视频生成配音：正常消耗

### 注意事项

- 非正脸/远景图片也可生成，效果依然出色
- 图片要求：清晰正脸效果更佳，但侧脸/全景也可识别
- 上传视频时需在输入框填写台词内容

## 四、动作模仿

**用途：** 让指定角色模仿参考视频中的动作。

### 启用方式

即梦首页 → 左侧生成 → 模式切换为**动作模仿**

### 模型选择

| 模式 | 特点 |
|------|------|
| 快速模式 | 生成速度快，成本低 |
| 生动模式 | 默认，平衡效果与速度 |
| 大师模式 | 画质更好，动作更流畅 |

### 操作方式

**方式 1：官方模板**
1. 上传角色图片（半身像/全身像）
2. 点击参考视频 → 选择动作模板（4 个基础模板）
3. 点击扩展可预览动作效果
4. 点击生成

**方式 2：自定义视频**
1. 上传角色图片
2. 上传本地参考视频
3. 点击生成

### 效果特点

- 全身动作：流畅度较好，可能存在肢体逻辑错误
- 半身动作：流畅度高，细节还原到位（包括手指运动）
- 分辨率：生动/快速模式约损失 50% 清晰度，大师模式有所提升
- 影子效果：随动作自然变化

### 后期优化

生成后可在视频查看页面使用：
- 提升分辨率
- 补帧

## 五、音效（剪映）

剪映 → 音频 → **音效库** → 搜索关键词（如"脚步声"、"拳击"、"雷声"）

音效选择依赖生活经验判断：
- 奔跑场景 → 脚步声
- 打斗动作 → 拳击/碰撞音效
- 戏剧转折 → 闪电/弦乐重音
- 人群场景 → 人群嘈杂声
- 海浪场景 → 海风海浪声
- 魔法效果 → 魔法发光音效
- 鸟鸣场景 → 鸟鸣音效
