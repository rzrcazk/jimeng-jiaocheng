# 运镜提示词

## 提示词公式

```
镜头类型 + 主体动作 + 环境描述 + 固定品质词
```

**固定品质词（视频通用后缀）：**
```
画面真实，镜头流畅，没有闪烁，生动细节，光线统一柔和，画面主体动态呈现，动作自然流畅，环境光，过程丝滑而匀速，符合逻辑、极致细节，超真实动态捕捉形，清晰自然，不模糊，高质量，没有瑕疵
```

## 一、10大基础运镜类型

| 运镜类型 | 英文 | 提示词关键词 | 效果/用途 |
|---------|------|------------|---------|
| 主观视角 | POV Shot | 第一人称镜头 | 沉浸感，"魂穿"主角 |
| 手持摄影 | Handheld Shot | 纪实感手持镜头；轻微晃动；前景 | 呼吸感、临场感 |
| 全景镜头 | Wide Shot | 史诗级全景 | 交代环境、营造氛围 |
| 特写镜头 | Close Up | 超写实；微距特写 | 放大细节、传递情感 |
| 中景镜头 | Medium Shot | 电影感中景 | 叙事主力，兼顾动作与环境 |
| 推轨镜头 | Tracking Shot | 电影级推轨镜头；平稳跟随 | 跟拍、推进、摇臂等动态镜头 |
| 过肩镜头 | OTS Shot | 过肩视角；前景；中景 | 增强叙事性和代入感 |
| 滑动变焦 | Dolly Zoom | 希区柯克式滑动变焦；推进镜头同时缩小焦距 | 强烈戏剧冲突或心理感受 |
| 深焦镜头 | Deep Focus | 电影级深焦镜头；前景/中景/远景 | 宏大场景、丰富细节 |
| 环绕镜头 | Orbit Shot | 镜头环绕；360度旋转 | 全方位展示主体 |

## 五、FPV 一镜到底

### 提示词模板

用豆包（或其他 LLM）生成 FPV 五段连贯提示词，公式：

```
角色：你是视频提示词专家，善于用一镜到底的镜头语言讲述故事。

请根据示例提示词格式，生成 [场景描述] 的视频提示词。
要求：
- 5个画面连贯（一镜到底）
- 上帝视角，无主角
- 每个镜头不超过80字
- 每个镜头里都必须有转弯
```

### 示例：废墟丧尸场景

**图片提示词（先生成场景图）：**
```

```

**视频提示词（5段FPV）：**
```
1.全景 FPV 高空俯冲：镜头极速推进穿过烟雾弥漫摩天大楼急转直下扎入布满废墟的街道，左侧废弃巴士旁三只丧尸正低头啃食残骸。

2.中景 FPV 转入小巷：镜头右转撞开锈蚀铁门，沿逼仄小巷推进，墙面褪色涂鸦被藤蔓覆盖，右侧垃圾桶翻倒滚落罐头，转角处突然晃过一个拖着断腿爬行的丧尸。

3.近景 FPV 穿越商铺：镜头穿过玻璃碎渣迸溅的橱窗，左转扫过货架上倾倒的药品，天花板钢筋坠落扬起灰尘，前方丧尸群正撞翻货架蜂拥而来。

4.特写 FPV 绕过断墙：镜头贴着布满弹孔的断墙侧移，右转时铁网围栏突然晃动，数十只丧尸隔着网抓挠嘶吼，其中一只腐烂手掌穿透网眼挥动。

5.拉升 FPV 环摇全景：镜头急遽上升并顺时针环摇，断裂高架桥在画面中央交叉，桥下丧尸群如黑色浪潮蔓延，远处残阳将废墟染成暗红，画面定格在扭曲的路牌"安全区→"。
```
