# 剪映剪辑与成片导出

## 剪映界面4区域

| 区域 | 位置 | 内容 |
|------|------|------|
| 素材区 | 左 | 素材导入、音频库、文本、贴纸、特效、转场、字幕 |
| 预览窗口 | 中 | 播放预览，可切换画质和比例 |
| 参数编辑区 | 右 | 选中素材后显示可编辑参数（大小/位置/旋转/变速等） |
| 剪辑轨道 | 下 | 控制视频顺序、时长、音频/特效位置关系 |

---

## 剪辑流程

### Step 1：导入素材

- 素材区 → 导入按钮，或直接拖拽到素材区/轨道
- 视频、音频（配乐/配音/音效）全部导入

### Step 2：按脚本调整顺序和时长

- 拖拽轨道上的视频片段调整顺序
- **分割操作**：将时间轴指针拖到目标位置 → 点击"分割"按钮 → 选中多余部分 → Delete键删除
- **恢复误删**：选中片段 → 鼠标拖拽片段尾端往后拖
- **变速**：选中片段 → 右侧参数区 → 变速（如爷爷走路太快 → 0.79倍速）
- **轨道缩放**：右下角滑块左右滑动，调整轨道显示密度（精细编辑时放大）

### Step 3：添加配音和配乐

- 音频文件导入轨道 → 拖到合适位置对齐画面
- 配乐过长：在视频末尾分割音频 → 删除多余部分
- 配乐前奏无声：把音频轨道往前拖，找到有声的起点

### Step 4：添加音效

- 素材区 → 音频 → 音效库 → 搜索关键词
- 找到合适音效 → 拖到对应画面位置的音频轨道

### Step 5：添加特效

- 素材区 → 特效 → 搜索关键词（如"闪白"、"光效"）
- 长按特效拖到轨道，调整时长以匹配画面节奏

### Step 6：添加转场

- **AI视频最常用转场**：叠化（两段视频衔接处，0.3～0.5秒）
- 素材区 → 转场 → 叠化 → 拖到两段视频的连接处
- 每段视频的转场需求不同，根据内容选择

### Step 7：字幕

- **自动识别**：素材区 → 字幕 → 开始识别
  - 剪映自动识别音频中的语音并生成字幕条
  - 校对识别错误的文字（右侧文本框修改）
  - 效率远高于手动添加

### Step 8：导出成片

- 右上角导出按钮 → 选分辨率和码率 → 导出

## 常用轨道操作

| 图标 | 功能 | 说明 |
|------|------|------|
| 🔒 锁 | 锁定轨道 | 防止该轨道素材被误编辑 |
| 👁 眼睛 | 隐藏轨道 | 预览时不显示该轨道内容 |
| 🔊 喇叭 | 静音 | 静音该轨道（不影响其他轨道） |

---

## 预览窗口说明

| 按钮 | 功能 |
|------|------|
| 画质选择（原画/高清/标清） | 调低画质提升预览流畅度，大工程文件时有用 |
| 比例选择 | 修改整体视频比例（改变后原素材四周会出现黑边） |
| 全屏 | 全屏预览 |

## 成片质量建议

- 音效要结合生活经验：每个画面判断"真实世界里这个场景有什么声音"
- 特效使用克制：只在转折/高潮点使用，避免滥用
- 字幕校对不可跳过：AI识别有误差，校对确保观看体验
- 转场叠化的时长影响节奏感：快节奏镜头用短叠化（0.2s），慢镜头用长叠化（0.5s+）
