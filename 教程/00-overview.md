# 即梦AI视频创作全流程概览

## 工具栈

| 工具 | 用途 |
|------|------|
| 即梦AI (jimeng.jianying.com) | 图片生成、视频生成、数字人（对口型） |
| 豆包 / DeepSeek | 剧本生成、角色提示词生成、配乐提示词生成 |
| 剪映 | 视频剪辑、字幕、特效、转场、导出 |
| 海螺AI (MiniMax) | AI配音——旁白与角色对白 |
| 魔兔AI | AI配乐生成 |

## 完整工作流

```
剧本创作 → 角色设计 → 分镜生图 → 视频生成 → 配音配乐 → 剪辑成片
  01        02          03           04           05          06
```

每个阶段对应一份独立教程文件。

## 即梦AI操作模式

顶部菜单切换四种模式：

| 模式 | 用途 |
|------|------|
| Agent模式 | 对话式生图/视频，适合快速原型 |
| 图片生成 | 专用图片生成界面，支持图生图 |
| 视频生成 | 含首尾帧/智能多帧/Seedance 2.0 |
| 数字人 | 上传角色图+音频，生成对口型视频 |

## 即梦积分说明

- 图片4.0：免费
- 视频生成：消耗积分；时长越长、帧数越多，消耗越多
  - 示例：智能多帧（5张图）≈ 40积分
- 每日登录赠送积分；积分耗尽可换其他平台（可灵、海螺、Runway等逻辑相同）

## 模型选择原则

- 图片：**图片4.0**（最新，效果最佳，免费）
- 视频：**视频3.0**（常规生成）/ **Seedance 2.0**（多角色参考、高级运镜）
- 配音：**海螺AI 2.5模型**（最新）
- 配乐：**魔兔AI最高编号模型**

## 常用分辨率与比例

| 场景 | 比例 | 分辨率 |
|------|------|--------|
| 横屏短片 | 16:9 | 720P（测试）/ 1080P（成片） |
| 竖屏短视频 | 9:16 | 720P（测试）/ 1080P（成片） |
| 角色三视图 | 16:9 | 2K（练习够用） |
| 角色单图 | 9:16 | 2K |

## 视频时长建议

- 单段视频：**5秒**（默认，复杂动作更稳定）
- 10秒可选，但复杂动态不合理概率增大，且消耗积分更多
- Seedance 2.0 参考视频：**≤15秒**
