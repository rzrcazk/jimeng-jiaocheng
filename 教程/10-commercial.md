# 商业应用

## 一、产品广告四步流程

### 步骤 1：获取提示词（豆包 AI）

**提示词：**
```
我现在需要制作一个水果奶茶的产品广告，给我生成 4 个分镜头图片的提示词，用于即梦 AI 生成图片。
镜头一是什么，镜头二是什么，镜头三是什么，镜头四是什么。
```

### 步骤 2：生成图片（即梦 AI）

1. 即梦 → 图片生成模式
2. 粘贴提示词
3. 设置尺寸：
   - 手机平台：9:16 / 2:3 / 3:4
   - 电脑平台：16:9
4. 点击生成

**参考图生图（保留产品造型）：**
1. 上传产品参考图
2. 修改提示词：`右侧摆放水果奶茶`（改为单数）
3. 点击图片 → 抠图 → 保存（锁定产品）
4. 生成

### 步骤 3：生成视频

**方法 1：自己描述运镜**
1. 即梦 → 视频生成 → 首帧图生视频
2. 提示词：`镜头环绕拍摄，水果上的水滴自然往下流动`

**方法 2：豆包 AI 反推运镜**
1. 上传整理好的图片到豆包 AI
2. 提示词：
   ```
   你是一个 AI 广告制作大师，经验丰富。给这几个图片分别书写一段用于视频生成的提示词，
   内容包括运镜和画面中事物的运动描述。
   ```
3. 复制提示词到即梦生成视频

### 步骤 4：剪辑成片（剪映）

1. 导入视频片段
2. 添加配音配乐（搜索"奶茶广告"）
3. 添加字幕文本（产品名称、宣传语）
4. 添加转场效果
5. 调整视频片段时长与音乐匹配

## 二、电商产品宣传图

### 方法 1：参考现有作品

1. 即梦首页搜索"电商产品"
2. 查看喜欢的作品，复制提示词
3. 豆包 AI 处理：`只保留上方提示词中和环境相关的描述，其余部分去除`
4. 即梦图片生成 → 上传产品图 → 抠图锁定 → 生成

### 方法 2：豆包 AI 原创

1. 上传产品图到豆包 AI
2. 提示词：
   ```
   根据产品图，给产品搭配一个电商宣传用的环境，提示词描述要求凸显自然温和清新，
   色彩搭配要和产品本身契合。
   ```
3. 复制提示词到即梦
4. 上传产品图 → 抠图 → 生成

### 多风格尝试

**豆包 AI 提示词：**
```
多来几个不同风格的。
```

## 三、丝滑转场电商视频

### 流程

**Step 1：准备分镜图片**
1. 豆包 AI 上传产品图
2. 提示词：
   ```
   以参考图为首帧图，根据内容生成多机位多视角的分镜头画面，提示词一共生成 5 个，
   展示产品外观，可以出现产品周围开着花朵的画面。分镜头画面描述结构为：景别 + 内容，
   用词简洁。
   ```
3. 即梦参考图生图生成各分镜
4. 下载并按顺序标号

**Step 2：生成衔接提示词**
1. 上传排序好的图片到豆包 AI
2. 提示词：
   ```
   我将使用即梦 AI 的智能多帧功能生成一段产品宣传广告，需要你根据上传的参考图顺序，
   给出图片之间衔接用的提示词，要求连贯过渡丝滑自然，用词简洁。
   ```

**Step 3：智能多帧生成视频**
1. 即梦 → 视频生成 → 智能多帧
2. 依次上传图片
3. 图片间添加衔接提示词
4. 调整各段时长（变化快的缩短，场景转换的延长）
5. 生成

### 创意转场增强

**豆包 AI 提示词追加：**
```
要求伴随着画面场景的变化，例如沙漠、雨林、草原等。
```

**手动描述转场：**
```
镜头环绕拍摄，场景逐渐变为沙漠，自然过渡
```
```
镜头自然移动，画面散落在香水周围
```

## 四、AI 美女博主视频生成

### 流程

**Step 1：反推提示词**
1. 即梦首页搜索"美女"
2. 下载喜欢的图片
3. 豆包 AI 上传该图片
4. 提示词：`反推这个图片的中文提示词，用于即梦 AI 文生图`

**Step 2：生成图片**
1. 即梦图片生成 → 粘贴提示词
2. 修改问题：如`唇色是温润的水红色` → 删除强调唇色的词
3. 调整尺寸（手机 9:16，电脑 16:9）
4. 生成

**Step 3：生成视频**
1. 即梦视频生成 → 首帧图生视频
2. 提示词：`镜头跟随人物后退，身后的旋转木马在转动，人群在走动`
3. 强调背景物体运动（避免被忽视）

**Step 4：剪辑成片**
1. 导入视频片段
2. 添加背景音乐（搜索"日常 vlog"）
3. 添加音效：
   - 人群场景 → 人群嘈杂声
   - 海滩场景 → 海浪声
   - 室内场景 → 可省略
4. 添加开场特效（如相机拍摄边框）

## 五、运营海报框架

### 设计构思三步法

#### Step 1：确定主题风格

1. 主题：购物节
2. 氛围：打折促销
3. 美术风格：3D 渲染卡通风格

#### Step 2：编写关键词

| 要素 | 内容 |
|------|------|
| 画面内容 | 推着购物车的人，家人拎着购物袋，卡通形象，明亮欢快表情，微笑，休闲服装 |
| 背景 | 商场购物街，礼物盒，购物卡，优惠券 |
| 布局 | 文字居中，人物位于画面中心面向观众 |
| 字体内容 | "购物狂欢节" + "We Shopping"（中英排版） |
| 字体设计 | 动感活力，有棱角，有层次感，现代醒目 |
| 色彩 | 明亮大胆，温暖的绿色橙色和粉红色 |
| 风格氛围 | 充满活力，卡通，有庆祝感 |
| 应用场景 | 数字广告，社交媒体 |

#### Step 3：生成海报

1. 即梦图片生成 → 粘贴完整提示词
2. 模型：4.1（最新版）
3. 比例：手机 2:3/9:16，电脑 16:9
4. 生成

### 修改配色

**问题：** 绿色不够喜庆

**解决：** 删除`重点用温暖的绿色`，重新生成

### 换主题（豆包 AI）

**提示词：**
```
保持上方的提示词框架不变，将主体以及相关内容替换为烤肉节，把修改好之后的提示词发给我。
```

## 六、字体设计框架

### 字体设计六要素

| 要素 | 内容 | 示例 |
|------|------|------|
| 文字内容 | 要生成的文字（加双引号） | `"夏日孤岛派对"` |
| 字体风格 | 斜体/涂鸦/流动变形等 | `流动变形体` |
| 字形描述 | 形态变化 | `波浪般的起伏` |
| 笔触描述 | 运笔技法/画布/笔刷 | `细微的颗粒感`、`油画亚麻布` |
| 材质描述 | 亚克力/树脂/水晶/金属/油漆 | `红橙色渐变玻璃质感` |
| 背景描述 | 建议纯色方便抠图 | `纯白色背景` |

### 获取字体风格列表（豆包 AI）

**提示词：**
```
字体设计中字体风格都有哪些，列举出 10 种常见风格。
```

### 完整示例

```
文字内容："夏日孤岛派对"
字体风格：流动变形体
字形描述：波浪般的起伏
笔触描述：细微的颗粒质感
材质描述：红橙色渐变玻璃质感
背景描述：夕阳下的海滩，有棕榈树（或纯白色背景方便抠图）
```

### 生成设置

1. 即梦图片生成
2. 点击**文字效果增强**（自动加双引号）
3. 比例：横向 16:9，纵向 2:3
4. 生成

### 豆包 AI 批量生成提示词

**提示词：**
```
帮我写几段字体设计的提示词，用于即梦 AI 的生成。
需要包括：文字描述、字体风格、字形描述、笔触描述、材质描述、背景描述。
其中背景描述最好是纯色背景，方便抠图。
```

## 七、城市创意海报

### 方法 1：做同款修改

1. 即梦首页搜索"创意城市"
2. 查看喜欢的作品，复制提示词
3. 点击**做同款**
4. 修改城市相关内容：
   - 上海 → 北京
   - 手机 → 青花瓷盘
   - 建筑：东方明珠 → 故宫/胡同/央视大楼

### 方法 2：豆包 AI 批量生成

**提示词：**
```
根据上述提示词，生成新的富有创意的微缩城市提示词，要求尽情发挥创意，不拘泥于上述提示词的元素。
```

### 文字主体海报

同样方法：搜索"城市 文字" → 复制提示词 → 豆包 AI 替换城市 → 生成

## 八、字体设计动态视频

### 流程

**Step 1：豆包 AI 生成多分镜提示词**
```
以这张图片作为起始镜头，根据内容生成一个多机位视频，呈现出壮观的氛围。需要 4 个连贯分镜，分镜头脚本结构为：景别 + 视角 + 运镜 + 画面内容，每个分镜用不同的运镜方式，要求用词简洁，字数不超过 400 字。
```

**Step 2：即梦生成视频**
1. 上传首帧图
2. 粘贴分镜提示词
3. 时长调整（4 分镜建议 10 秒）
4. 生成

**Step 3：AI 配乐（可选）**
1. 视频页面 → AI 配乐
2. 自定义配乐：场景/流派/情感/乐器
3. 生成 3 段选择
