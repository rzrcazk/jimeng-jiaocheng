# 视频生成

## 基础设置

模式：**视频生成**

| 参数 | 常用值 | 说明 |
|------|--------|------|
| 模型 | 视频 3.0 / Seedance 2.0 | 常规生成用视频 3.0；多角色高级参考用 Seedance 2.0 |
| 比例 | 16:9 | 横屏；9:16 竖屏 |
| 分辨率 | 720P | 测试用；成片用 1080P |
| 时长 | 5 秒 | 默认；复杂动作优先选 5s；跨度大用 10s |

---

## 一、文生视频

**用途**：找风格灵感，不适合直接用于成片（风格不可控）。

**提示词 6 要素：**
```
主体 + 背景 + 风格 + 色调 + 氛围 + 运镜
```

示例：
```
米黄色古代朴素服装的小女孩，古树林背景，3D 风格迪士尼动画融入中式美学，
蓝绿色为主棕黄色为辅，安逸悠然古朴氛围，固定镜头人物正走向椅旁的古树
```

---

## 二、图生视频（首尾帧）

**最常用方式**，可控性强，风格贴近参考图。

**操作：**
1. 视频生成界面 → 确认首尾帧模式（默认）
2. 输入框左侧两个带加号的小框 = 首帧 / 尾帧
3. 上传图片到**首帧**
4. 提示词只需描述**运动和运镜**，不需要重复描述画面内容

**提示词示例：**
```
跟随镜头，女孩跑来蹲在地上看小草
```
```
镜头紧紧跟随，人物向前移动
```
```
固定镜头，人物走向远处
```

---

## 三、一镜到底长镜头

通过逐段生成 + 帧接续，拼接超过 5/10 秒的长镜头。

**步骤：**
1. 用首帧图片生成第一段（5 秒）→ 下载
2. 导入剪映，将时间轴拖到最后一帧 → 全屏截图最后一帧
3. 回到即梦，将截图替换为**新的首帧**，提示词不变 → 生成第二段
4. 下载第二段，导入剪映拼接到第一段后面
5. 在两段连接处添加**叠化**转场（0.3～0.5 秒）
6. 重复步骤 2-5，直到达到目标时长

---

## 四、连贯镜头（智能多帧）

从多张静态分镜图生成连贯的动态视频。

**操作：**
1. 视频生成界面
2. 输入框左下角第一个按钮 → 切换为**视频生成**
3. 第三个按钮（首尾帧）→ 切换为**智能多帧**
4. 依次上传 5 张按顺序排列的分镜图（带加号小框依次上传）
5. 两张图片中间可填写镜头转换提示词（可选，不填则 AI 自由发挥）
6. 点击发送；确认积分消耗（约 40 积分/5 帧）后生成

**效果：** 人物一致性好，动态连贯自然。

## 五、Seedance 2.0（多角色高级参考）

适合：多角色互动场景、自定义声音、精确分镜控制。

### 启用方式
视频生成 → 模型选择 **Seedance 2.0** → 参考方式选**全能参考**（非首尾帧）

### 参考素材准备

| 参考类型 | 要求 |
|---------|------|
| 角色图（可多个） | 人物照片或设计图 |
| 音色参考 | ≤14 秒音频（可自录） |
| 场景图 | 单张场景图片 |
| 分镜参考视频 | ≤15 秒（超过需剪辑切分） |
| 武器/道具图（可选）| 单张图片 |

### 核心约束

- 图片 ≤9 张 + 视频 ≤3 个，总文件数 ≤12
- 生成时长：4-15 秒自由调节
- 延长视频功能：上传已有视频，提示词描述"延长 5 秒"，最多延长至 15 秒
- 翻唱功能：上传人物图 + 歌曲名，生成对口型翻唱视频
- 首尾帧循环：首尾帧使用相同图片，生成无缝循环视频
- 参考视频必须 **≤15 秒**，超过需提前在剪映剪成多段
- 全能参考模式是 Seedance 2.0 专属，其他模型无此选项
- 支持商业广告（产品多角度图→广告视频）、仙侠剧情、打斗场景等多种题材
- 图片未清晰说明用途时，模型可能重复使用产生误导

### 提示词格式

```
男主角参考 @[角色 A 图片]
女主角参考 @[角色 B 图片]
场景参考 @[场景图片]
画面分镜参考 @[分镜视频]
男主角声音参考 @[音频文件]

男子手持长枪，@[女角色]，女子与手持双刀的女子激烈打斗，特效华丽，打斗镜头切换丝滑流畅
画面写实风格逼真

0-2 秒：男子冲向女子（全景镜头）
3-15 秒：分镜参考 @[分镜视频]
```

**（可选）武器参考：**
```
女主角武器参考 @[武器图片]
```

### 风格控制

在提示词中可同时指定风格：
```
画面写实风格逼真，场景转换成写实风格
```
或：
```
画面动漫风格，参考人物风格
```

## 六、首尾帧景别匹配策略

景别差异决定运镜描述方式：

| 景别关系 | 运镜策略 | 提示词示例 |
|---------|---------|-----------|
| 景别相同（高度一致） | 平移/旋转 | `镜头向左平移，自然过渡到下一张图片` |
| 有高度差（俯拍→中近景） | 上升/下降 | `降镜头，镜头快速下降，自然过渡到下一个图片` |
| 跨度大（室外→室内/酒馆→农场） | 完整描述运镜过程 | `镜头快速上升后推进，越过城市，随后下降到一处农场` |
| 时间流逝（日落→月夜） | 延时摄影 | `斗转星移，延时摄影，月落日升` |
| 创意转场（水面→水下异空间） | 想象驱动 | `镜头从水面进入水中，自然过渡到下一个图` |

**时长建议：**
- 5 秒：景别相同或小跨度转场
- 10 秒：场景跨度大、需要充足过渡时间

## 七、豆包 AI 辅助反推运镜提示词工作流

**单段运镜反推：**
1. 上传两张图片到豆包 AI（注意先后顺序）
2. 提示词：
   ```
   我将使用这两张图片作为首尾帧，生成一段 5 秒的视频。
   根据画面内容生成对应的运镜提示词，要求用词简洁且转场过渡自然，
   用一句话概括。
   ```
3. 复制生成的运镜提示词到即梦
4. 如需更精简，追加要求：`提示词内容再精简一些，尽量一句话内说明`

**智能多帧衔接提示词：**
```
我将使用即梦 AI 的智能多帧功能生成一段产品宣传广告，
需要你根据上传的参考图顺序，给出图片之间衔接用的提示词，
要求连贯过渡丝滑自然，用词简洁。
```

## 八、视频延长功能

**操作：**
1. 上传已有视频到参考窗口
2. 提示词描述：`@[视频 1] 延长 5 秒`
3. 最多可延长至 15 秒

**注意：** 即梦目前视频延长功能有限，可灵、Runway、MiniMax 等平台也有此功能。

---

## 九、首尾帧循环视频

**用途：** 课件 PPT 循环播放、背景视频等需要无缝衔接的场景。

**操作：**
1. 首帧和尾帧上传**相同图片**
2. 提示词描述画面动作（如"人物抬头看太阳，两孩童互相交谈"）
3. 时长设为 10 秒（5 秒会导致动作幅度过小）
4. 生成后得到无缝循环视频
