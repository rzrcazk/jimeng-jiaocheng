# 分镜画面生成

将脚本中每个镜头的画面描述转化为静态图片，作为后续视频生成的素材。

## 方法一：豆包图像生成（免费）

### 基础操作

1. 打开豆包 → 图像生成功能
2. 将脚本镜头的**画面内容字段**直接粘贴为提示词
3. 末尾追加风格描述（与角色设计保持一致）：
   ```
   3D风格，迪士尼动画风格，融入中式美学
   ```
4. 比例：16:9
5. 点击发送，从4张结果中选取满意的

### 保持后续镜头的一致性

生成第一个镜头后，后续镜头不新开对话，在同一对话中继续输入：
```
参考图片中的风格和人物形象，若画面中要出现人物的话，则需要保持人物特征和服饰的一致。
继续生成镜头[N]：[粘贴镜头N的画面内容]
```

或上传一张已选定的参考图（通过输入框下方的"参考图"按钮），然后在前面加：
```
参考图片中的风格和人物形象，若画面中要出现人物的话，则需要保持人物特征和服饰的一致，
继续生成镜头[N]
```

## 方法二：即梦4.0批量生成

适合一次生成多个分镜、且需要保持人物高一致性的场景。

### 设置

- 模式：图片生成
- 模型：图片4.0
- 比例：16:9

### 操作

1. 通过输入框左下角的带加号小框，上传各角色的**正视图**（已生成的三视图/标准站立图）
2. 提示词格式：
   ```
   图一是[角色A名称]，图二是[角色B名称]
   请根据以下内容生成[N]个分镜画面，要求人物特征服饰均保持一致，
   风格为3D风格，迪士尼动画风格，融入中式美学

   镜头1：[画面内容]
   镜头2：[画面内容]
   ...
   镜头N：[画面内容]
   ```
3. 即梦4.0单次最多生成40张图，可一次提交多个镜头

### 注意事项

- 一次生成可能不是每张都符合要求，**建议多生成几组**后挑选
- 人物特征与上传的参考图相似度较高
- 复杂场景（多角色互动、特殊构图）也能较好消化

---

## 图片下载与整理

1. 点击满意的图片 → 右上角下载按钮（下载原图）
2. 按镜号命名文件，导入到分镜脚本文档中对应位置
3. 所有角色分镜图集齐后，进入视频生成阶段
